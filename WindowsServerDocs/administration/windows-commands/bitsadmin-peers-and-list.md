# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers and list

Répertorie tous les homologues.

## <a name="syntax"></a>Syntaxe

```
bitsadmin /peers /list
```

## <a name="examples"></a>Exemples

Pour afficher la liste de tous les pairs :

```
bitsadmin /peers /list
```
