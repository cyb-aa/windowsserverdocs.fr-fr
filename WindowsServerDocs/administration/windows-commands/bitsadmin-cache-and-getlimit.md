# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache and getlimit

Récupère la limite du cache.

## <a name="syntax"></a>Syntaxe

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a>Exemples

Pour récupérer la limite du cache :

```
bitsadmin /cache /getlimit
```
