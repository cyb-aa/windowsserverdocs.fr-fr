# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache and clear

Purge le cache local.

## <a name="syntax"></a>Syntaxe

```
bitsadmin /cache /clear
```

## <a name="examples"></a>Exemples

Pour purger le cache local :

```
bitsadmin /cache /clear
```
