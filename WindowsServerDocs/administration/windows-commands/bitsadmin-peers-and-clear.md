# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers and clear

Efface la liste d’homologues.

## <a name="syntax"></a>Syntaxe

```
bitsadmin /peers /clear
```

## <a name="examples"></a>Exemples

Pour effacer la liste d’homologues.

```
bitsadmin /peers /clear
```
