# <a name="required-updates-for-active-directory-federation-services-ad-fs-and-web-application-proxy-wap"></a>Mises à jour requises pour Active Directory Federation Services (ADFS) et Proxy d’Application Web (WAP)


À compter d’octobre 2016, toutes les mises à jour à tous les composants de Windows Server sont libérés uniquement via Windows Update (WU).  Il existe des téléchargements individuels ou aucun correctif plus.
Cela s’applique à Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 et Windows Server 2008 R2 SP1.

Cette page répertorie les packages de cumul d’intérêt particulier pour AD FS et WAP, ainsi que la liste d’historique des mises à jour de correctif recommandé pour AD FS et WAP.

## <a name="updates-for-ad-fs-and-wap-in-windows-server-2016"></a>Mises à jour pour AD FS et WAP dans Windows Server 2016
Mises à jour pour Windows Server 2016 sont remis à tous les mois via Windows Update et sont cumulatifs. Le package de mise à jour répertorié ci-dessous est recommandé pour tous les serveurs AD FS et WAP 2016 et inclut toutes les mises à jour précédemment requises, ainsi que les derniers correctifs.
